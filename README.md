# Deal Screener

An interactive single-family real estate deal screener for buy-and-hold and
fix-and-flip underwriting. Enter a property's numbers and get an instant
go / no-go read against an editable buy box, with built-in stress-testing.

Built by **Clave Concept** as a demonstration of operations and AI build capability.

## What it does
- **Two modes:** buy-and-hold rental analysis and fix-and-flip analysis, in one view.
- **Buy-and-hold metrics:** monthly cash flow, cash-on-cash return, cap rate,
  net operating income, and a 70%-rule maximum offer.
- **Fix-and-flip metrics:** net profit, return on cash, annualized return,
  total project cost, and cash invested.
- **Real financing math:** standard mortgage amortization driven by down payment,
  interest rate, and loan term.
- **Editable buy box:** set your own thresholds and the tool returns a
  Pursue / Analyze / Pass verdict.
- **Pressure test:** sliders flex after-repair value, rehab, and rent so you can
  stress any deal live.

## Note
This is a demonstration model. All figures shown are sample data for illustration

decision.

## Tech
